**A guide to run the application**
Open Terminal -> slit terminal
1. Open server:
    in first terminal do the following:
    cd Backend -> cd LSA -> cd server -> python server.py
2. Open client:
    in second terminal do the following:
    cd Frontend -> cd dexStarter -> cd dex -> npm start
